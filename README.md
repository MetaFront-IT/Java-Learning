یک پروژه برای تکالیف کلاس **Java**.\
این پروژه با زبان **Java** نوشته شده.
نام: آرشان لطفی.
کلاس:جمعه 9 تا 13.

---



## 🚀 نحوه اجرا

### 1. کلون کردن مخزن

```bash
git clone https://github.com/MetaFront-IT/Java-Learning.git
```

### 2. ورود به پوشه پروژه

```bash
cd Java-learning
```

### 3. کامپایل و اجرای برنامه

```bash
javac filename.java
java filename
```

> **نکته:** توجه داشته باشید که شما باید از JDK-25 استفاده کنید.\
